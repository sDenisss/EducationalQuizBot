
Edu: Переменные в C# — это именованные области памяти, которые могут хранить данные определённого типа. 
Тип переменной указывается при её объявлении. Например:
int number = 10;
string name = "Иван";

Edu: В C# ключевое слово var позволяет компилятору автоматически выводить тип переменной на основе присваиваемого значения.
Например:
var age = 25; // тип int
var message = "Привет"; // тип string

Edu: В языке C# есть несколько областей видимости переменных:
- Локальные переменные (доступны только внутри метода или блока кода)
- Поля класса
- Параметры метода
- Переменные уровня блока (внутри фигурных скобок)

Edu: Ключевое слово static в C# указывает, что член класса принадлежит самому классу, а не его экземпляру.
Статические члены можно вызывать без создания объекта:
Console.WriteLine(Math.PI);

Edu: В C# есть сборщик мусора (Garbage Collector), который автоматически освобождает память, удаляя неиспользуемые объекты.
Тем не менее, важно освобождать ресурсы вручную, реализуя IDisposable или используя оператор using.
